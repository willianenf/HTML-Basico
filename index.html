Separadores

Caso queira quebrar a linha digite <br>
Caso queira parágrafo digite <p>
Caso queira uma linha horizontal digite <hr>
Vale ressaltar que tags como <br> <hr> são tags nulas, ou seja não se fecham.

Formatação dos textos

Caso queira deixar em negrito (bold) digite <b>
Por exemplo: Lá ele
Caso queira deixar em negrito e representar algo importante, sério e urgente, digite <strong>
Por exemplo: Olha ele
Caso queira em Itálico digite <i> , usada em termos técnicos.
Por exemplo: O nome da flor é Orquídea.
Caso queira em Itálico com enfâse em uma palavra digite <em> deve ser usada para dar ênfase a uma palavra.
Por exemplo: Ela não passou por aqui ontem. E na acessebilidade o texto é lido com mais intensidade.
Caso queira em sublinhado digite <u>
Por exemplo: Norman
Caso queira o texto subscrito (letras pequenas abaixo de um texto) digite <sub>
Por exemplo: H20.
Caso queira o texto em sobrescrito (letras pequenas acima de um texto) digite <sup>.
Por exemplo: X 4.

Ps: Não são tags nulas.

Fazer Formulário

Para fazer um formulário digite <form>
A melhor prática sempre definir pelo menos o atributo action e o atributo method.
O atributo action define o local (uma URL) em que os dados recolhidos do formulário devem ser enviados.
O atributo method define qual o método HTTP para enviar os dados (ele pode ser "GET" ou "POST").
GET: Quando clicamos em um link da tag <a>, o link utiliza o método GET. O método GET possui capacidade para 1.024 caracteres e é utilizado para passar pequenas informações ao servidor. Ele vem pôr padrão.
POST: O método POST não retorna ao cliente. Desse modo, é considerado um método mais seguro. Além disso, o método POST não possui limite de capacidade, o que permite passar informações mais longas
O atributo type da tag <input> é o mais importante. Tal atributo define como o elemento deve comportar-se.
Também inserimos uma tag <textarea> para verificar a diferença dela com a tag <input> do tipo text.
Por fim, inserimos uma tag <label>para cada elemento, a fim de facilitar a identificação de cada um.

Checkbox

Checkbox é uma tag codificada para uma seleção múltipla e independente de opções(caso uma selecionada, não cancela a outra opção.)
Name e value são responsáveis por informar actions aos dados que devem ser enviados ao servidor.
Para adicionar checkbox basta colocar <input type="checkbox" name="nome" value="SelectA">
<input type="checkbox" name="nome" value="SelectB">
Caso coloque checked, o elemento já aparecer selecionado.
