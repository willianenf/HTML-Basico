Extensões e Atalhos VSCode

Extensões

advanced-new-file permite criar um novo arquivo utilizando somente o teclado. Basta clicar "Ctrl+Alt+N"

file utils permite renomear, deletar, copias e afins um arquivo usando o teclado.

Atalhos

Ctrl + P : busca pelos arquivos do projeto.
Ctrl+Shift+P (ou F1) : Com o texto requerido selecionado, digite abb deve aparecer Wrap Abbreviation, selecione e digite a tag que você quer.
Ctrl+F: Ajuda a achar palavra no código, clicando Enter ela vai indo de cima para baixo.
Ctrl + D : Seleciona a próxima ocorrência do que está selecionado.
Ctrl + ''(Botão das aspas): Abre o terminal (se estiver fechado)| Foca ele caso não esteja | Fecha o terminal se estiver focado.
Shift+Tab: corrigi espaçamento resolvendo identação rapidamente.
Tab: identa texto selecionado.
Se você digitar uma tag com * e quantidade que você quer que repita, ela multiplica até onde foi pedido. Por exemplo: li*7
Se quiser ir mais além digite como exemplo: ul>li*5 ou ol>li*5.
Mudar tags de uma vez: Clica no que quer mudar pressionando Alt, solta e apaga, depois clica Alt de novo, que sai.
Se selecionado dentro de um parâmetro e você quiser ir para dentro de outro use o TAB.
