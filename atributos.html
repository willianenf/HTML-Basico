Explicando os valores dentro dos atributos

O atributo target suporta os seguintes valores:

_Blank : carrega o novo endereço de destino em uma nova janela ou uma nova aba.
_Self : abre o link na janela ou frame atual, continua navegando no site (padrão)(quer dizer na própria aba).
_Top : vai desfazer todos os frames e abrir o destino no navegador completo.
_Parent : similar ao uso do _top em uma referência à janela mãe nome-do-frame caso esteja usando frames, indicar o nome da janela a abrir.

_Top e _Parent são técnicas em quase desuso.

O atributo rel (relativo a) suporta os seguintes valores:

Next(Próximo) : indica que o link é para a próxima parte do documento atual.
Prev (Previous) : indica que o link é para a parte anterior do documento atual.
Author : indica que é um para o site do autor do artigo atual.
External : indica que é um link para outro site que não faz parte do site atual.
No follow : indica que é um link para um site não endossado, como um link pago.
Link Local ou Link interno : ele leva o visitante a outra página dentro do nosso próprio site.
Link Externo : leva para fora do site, com o uso da URL completa, incluindo protocolo http:// ou https://

Download e Type

O uso do download e o type está ligado em dependendo do seu navegador ele já solicite o download do arquivo.
E você pode clicar com o lado direito do mouse e escolher salvar link "como".
